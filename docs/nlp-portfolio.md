# NLP Portfolio: Web Mining and Applied NLP

**Author:** Katie McGaughey  
**Course:** Web Mining & Applied NLP  

---

## 1. NLP Techniques Implemented

- **Tokenization**
  - Used in `stage03_transform.py` to split text into tokens.
- **Text Cleaning and Normalization**
  - Lowercasing, punctuation removal, and whitespace cleanup.
- **Frequency Analysis**
  - Used `Counter(tokens)` to compute top word frequencies.
  - Output: `data/processed/top_tokens.png`
- **Web Scraping**
  - Extracted HTML content using BeautifulSoup in `stage01_extract.py`.
- **API-Based Text Processing**
  - Parsed JSON data in Module 4 project.

---

## 2. Systems and Data Sources

- **HTML Web Pages**
  - Extracted text from arXiv articles.
- **JSON API Data**
  - Processed structured API responses.
- **Text Data**
  - Cleaned and analyzed raw text datasets.

**Handling messy data:**
- Removed HTML navigation noise
- Handled missing JSON fields
- Normalized inconsistent formatting

---

## 3. Pipeline Structure (EVTL)

- **Extract**
  - Pulled data from web pages and APIs using requests.
- **Validate**
  - Checked for required fields and HTML structure.
- **Transform**
  - Cleaned text, tokenized, and created features.
- **Load**
  - Saved outputs to CSV and generated visualizations.

---

## 4. Signals and Analysis Methods

- Word frequency using `Counter`
- Bigram analysis for context
- Token counts and unique word counts
- Visual outputs like charts and word clouds

---

## 5. Insights

- Cleaning significantly improves analysis quality
- Bigram analysis provides better context than single words
- HTML data requires more preprocessing than JSON
- Pipelines can be reused across different data sources
